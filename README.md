# dotfiles

My personal dotfiles, powered by https://chezmoi.io

## New system setup

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install chezmoi
chezmoi init linuswillner
chezmoi apply
```
